# E-Commerce Website

This is a simple e-commerce website built using HTML, CSS, and JavaScript. The project allows users to browse products, view product details, add items to the cart, and experience a responsive shopping interface.

## Features

- Product listing
- Product detail page
- Add to cart
- Remove items from cart
- Simple and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Download or clone the repository.

```bash
git clone https://github.com/rajatrajput121020-prog/E-commerce-website.git
```

2. Open the project folder.

```bash
cd ecommerce-website
```

3. Open the `index.html` file in your browser.

You can also use the Live Server extension in VS Code for a better development experience.
